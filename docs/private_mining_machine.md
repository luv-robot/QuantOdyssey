# Private Mining Machine

QuantOdyssey Phase 1 is a private AI research machine, not a public arena or trading product.

The near-term goal is to prove that the system can autonomously produce research findings worth
human attention:

```text
Research Harness
-> ResearchTask
-> strategy variants
-> backtest
-> optimizer / sensitivity
-> Monte Carlo / stress
-> Reviewer
-> next ResearchTask
```

Autonomy is allowed in research, not capital allocation:

```text
Autonomous research loop.
Human-gated promotion.
Zero autonomous capital allocation.
```

## Phase Order

```text
Private Mining Machine
-> Private Evaluation OS
-> Private-first Tooling
-> Community
-> Arena
-> Commercialization
```

Arena, ZK proof, community mechanics, and commercial packaging are intentionally downstream. They
only matter after the private system can generate and review meaningful strategy research.

## MVP Acceptance

Useful near-term acceptance criteria:

```text
weekly ResearchTask count >= 20
non-trivial ResearchTask count >= 5
weekly completed experiments >= 50
each strategy family has matched baselines
each strategy records search budget
Reviewer clusters repeated failures
Harness turns failures into next tasks
1-2 variants reach paper watchlist quality
```

If these are not true, platformization is premature.
